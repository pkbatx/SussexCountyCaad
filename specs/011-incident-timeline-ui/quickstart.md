# Quickstart: Incident Timeline Console

## Scenario 1: Unified Timeline Rendering

1. Open the incident detail view for an incident with multiple calls and transcripts.
2. Verify the timeline shows all events in chronological order with nested calls beneath their dispatch events.
3. Expand a timeline entry and confirm audio, transcript, and key facts are visible inline.

## Scenario 2: Playback Synchronization

1. Start playback on a timeline event.
2. Seek the waveform and confirm the transcript highlight and timeline focus update within 2 seconds.
3. Select a different timeline event and confirm the playback cursor jumps to the new timestamp.

## Scenario 3: Summary Evidence Linking

1. In the persistent summary panel, select a summary statement.
2. Verify supporting transcript lines and audio segments are revealed and highlighted in the timeline.
3. Confirm the summary updates when new timeline events arrive.
4. If a summary statement has no evidence, confirm the UI reports missing transcript data without errors.
