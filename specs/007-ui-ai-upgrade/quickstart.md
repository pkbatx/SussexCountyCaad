# Quickstart Scenarios

1. Ingest a call with a known agency and address; verify incident card shows normalized agency, type, address/town.
2. Toggle call list; verify call rows show agency, type, timestamp, incident association.
3. Play audio from a call row and call detail; verify player persists across navigation.
4. Apply agency filter; verify left column counts, center list, summary strip, and map update.
5. Open an incident detail; verify rollup history and key fields render cleanly.
6. Submit feedback (wrong location); verify UI updates immediately and feedback shows queued status.
7. Confirm insights strip lists active agencies and hotspots for the selected window.
8. Toggle map markers vs heatmap; verify map reacts to filters without hiding unmapped items.

## Validation Checklist

- [ ] Scenarios 1–8 validated locally
- [ ] Feedback queueing visible in stages or logs
- [ ] Insights panel populated for filtered window
- [ ] Map toggle updates without errors
